# 登录系统（Vue + SpringBoot）
本项目是前后端分离的登录系统，包含前端 Vue 页面和后端 SpringBoot 接口，实现用户登录验证功能。

## 项目结构
- loginvue/：前端 Vue 项目代码
- springboot-login-master/：后端 SpringBoot 项目代码

## 环境要求
- 前端：Node.js ≥ 16、Vite
- 后端：Java 11、MySQL 8.0
- 依赖管理：Maven（后端）、npm（前端）

## 运行方式
### 前端
```bash
cd loginvue/loginvue
npm install
npm run dev
